canvas

##canvas是一个画布，默认大小是300*150

如果想改变大小必须通过html的width，height属性增加像素改变大小，不能直接通过css设置，css设置会直接拉伸像素大小，并不能改变画布大小

##想要在canvas上绘图，必须得到canvas绘图工具

var ctx = canvas.getContext("2d");   2d 平面绘图    webgl 3d绘图
ctx.moveTo(100,100);//将绘图工具定点到100,100位置
ctx.lineTo(200,100);//从刚才位置绘制到哪里
ctx.stroke();描边绘图，将绘制的图形呈递出来

##ctx.fill()

调用fill方法，会将所有的点连接起来，即使没有结束的那条线，也会构成一个封闭的结构

###ctx.closePath() 

闭合路径方法，可以将结束点与开始点连接起来
ctx.lineWidth = num：用来设置线条的宽度  num是宽度

###ctx.beginPath() 

在canvas中开启新的内容，不影响上一次绘制

###虚线

ctx.setLineDash(arr)   arr中存在着实线部分和虚线部分

###颜色

ctx.strokeStyle= 'red';  设置描边颜色
ctx.fillStyle= 'red';  设置填充颜色

##绘制矩形

如果不设置style，不需要设置beginPath
ctx.rect(x,y,width,height);   虚描，需要描边  x,y起始点
ctx.strokeRect(x,y,width,height);   直接画矩形
ctx.fillRect(x,y,width,height);   填充矩形
ctx.clearRect(x,y,width,height); 清除绘制的矩形区域,橡皮擦功能
ctx.clearRect(0,0,cas.width,cas.height);清除整个画布
cas.width = cas.width;  相当于把画布刷新，也可以清除整个画布

##绘制圆弧

ctx.arc(x,y,r,startAngle,endAngle,clockWise);   需要描边呈现
x,y圆心坐标;r圆弧半径；startAngle圆开始角度；endAngle圆结束角度(需要转换为弧度) ；Math.PI = 180度   radian = angle * Math.PI /180；

##绘制文本

ctx.strokeText(文本，x,y);  文本字符串的左下角在x,y坐标处；文本内容空心
ctx.fillText(文本，x,y);   实心文本内容
ctx.font = '30px 黑体';   设置字体样式  大小、字体
ctx.textAlign水平对齐 取值left center right ； start end 默认为start  
ctx.textBaseline基线对齐 取值 top  middle bottom alphabetic(基线对齐)
ctx.measureText(文本).width；获取当前文本的宽度

##图片绘制

1.创建图片对象  var  img = new Image(); 
2.img.src = '...'
3.1 ctx.drawImage(img,x,y);  将图片绘制到x,y的位置
3.2 ctx.drawImage(img,x,y,width,height); 将图片绘制到矩形区域，
进行计算，等比缩放，
3.3 ctx.drawImage(img,sx,sy,sw,sh,x,y,w,h); 将图片的某个部分
绘制到canvas中，一般截取的部分宽高与绘制的宽高一致
drawImage不仅可以绘制图片，也可以绘制其他其它标签，例video、canvas等

##变换

变换操作的是坐标轴，当坐标轴发生了变化以后，坐标轴内的图形也会跟着变化
ctx.translate()  平移变换
ctx.rotate()  旋转变换
ctx.scale()   伸缩变换

##canvas状态

canvas中，凡是设置属性，都会延续到下一次修改。
ctx.save()  将当前状态保存   代码开始使用
ctx.restore()  将保存的状态恢复   代码结束使用
save 和 restore搭配使用，可以使上一次操作不影响下一次的

##什么是模块化
模块化是一种将系统分离成独立功能的方法 ，可将系统分割成独立的功能部分，
严格定义模块接口、模块之间具有透明性。

###模块化的优点
1.代码组织起来更加清晰，提升开发效率
2.减少了意外的bug，便于后期维护
3.避免污染全局作用域，减少了命名冲突

###如何实现模块化
1.将变量写在函数里面
2.将变量使用对象进行管理
3.使用闭包

###模块化规范
1.浏览器端规范
amd：(https://github.com/amdjs/amdjs-api/wiki/AMD) 代表：require.js
cmd: (https://github.com/cmdjs/specification/blob/master/draft/module.md) 代表：sea.js
2.服务器端规范
commonjs:(http://wiki.commonjs.org/wiki/CommonJS) 代表：node.js
ECMAScript2015(ES6)模块化规范

##seajs使用

###cmd规范特性
1.一个模块就是一个单独的文件
2.每个模块都是属于define关键字的回调函数被调用，每个模块都是一个单独的作用域
3.预加载、懒执行(预加载就是要先把所有的模块加载完成，懒执行就是需要执行那个模块才会执行那个模块)

###如何加载模块
1.在html文件中引入sea.js库(假设html同级的js文件夹中存放在js文件)
2.在页面script中使用seajs.use加载主模块，起名一般叫main.js

#逻辑运算符

1.短路运算符   a||b   a&&b  表达式的值等于决定表达式结果的值。
对于a||b，正所谓一true皆true，其实决定权在a中，首先判断a是否为真，如果为真则此时表达式等于a，如果a为假就等于b。常用在参数的处理
对于a&&b，一false皆false，开始决定权在a中，首先判断a是否为真，
如果为假则a决定了表达式，此时表达式值等于a，如果a为真，则移交决定权给b，
此时无论b是否为真，表达式都等于b的值。   常用于回调函数调用时处理

面向对象笔记

##面向对象性能优化

 1.有关for循环：
 当循环取值时，每次都要重新获取数组长度，会降低代码，尤其是操作HTMLCollections，需要实时查询DOM ，而DOM操作一般是比较昂贵的
 ，因此，我们在循环取值时可以缓存数组或集合的长度，这样只需查询一次，可以提高代码运行效率；
 例：for (var i = 0, max = myarray.length; i < max; i++) { 
     // 使用myarray[i]做点什么}

 2.能力检测：
 在框架加载时判断浏览器是否支持某些功能，保存在变量中，以后使用时判断变量即可，减少原型链的搜索，可以提高性能。
 判断浏览器是否支持某些方法可以进行方法能力检测或方法定义检测。
  方法定义检测：浏览器的内置方法都有类似的结构；{ [native 可以用正则表达式: /\{\s*\[native/ 判断
 例：function querySelectorAll() { [native code] }
 /\{\s*\[native/.test(document.querySelectorAll);   true支持/false不支持

 3.检测是否含有该类名：  cName是要匹配的类名
 将元素和匹配的类名的类名前后都加上一个空格，这样元素中的每一个类名前后都是以空格分开的，如果元素含有要匹配的类名，通过indexOf方法可返回
 一个大于-1的索引，如果没有，就返回-1；
 例：var eles = document.getElementsByTagName('*'),i;
 for(i = 0; i < eles.length; i++){
     if((' ' + eles[i].className + ' ').indexOf(' ' + cName + ' ') >-1){
         alert('匹配成功');
     }else{
         alert('匹配失败');
     }
 }

 4.Object.prototype 的成员
 1) constructor
 2) hasOwnProperty 判断该属性是否为自己提供
 3) propertyIsEnumerable 判断属性是否可以枚举
 4) isPrototypeOf    判断是否为原型对象
 5) toString, toLocaleString, valueOf

 5.documentfragment对象表示内存节点,在页面上不占位置
 创建节点通过  var dfg = document.createDocumentFragment();

 6.添加事件监听
 标准API   dom.addEventListener('click',function(e){...});
 IE低版本  dom.attachEvent('onclick',function(e){...});   该事件执行顺序是倒着的，先添加的会后执行

 7.获取原生的所有事件的方法
 var dv = document.createElement('div'),results = [];
 for(var k in dv){if(k.slice(0,2) == 'on'){results.push(k)}}
 所有的事件名称都保存在results中了。
