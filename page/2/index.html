for (var i = 0; i < num; i++) {
str += ‘0’;
}
return str;
}
str = str.substring(0, str.lastIndexOf(‘.’) + num + 1);
var start = str.substring(str.lastIndexOf(‘.’) + 1).length;
for (var j = start; j < num; j++) {
str += ‘0’;
}
return str;
};

seajs

##什么是模块化
模块化是一种将系统分离成独立功能的方法 ，可将系统分割成独立的功能部分，
严格定义模块接口、模块之间具有透明性。

###模块化的优点
1.代码组织起来更加清晰，提升开发效率
2.减少了意外的bug，便于后期维护
3.避免污染全局作用域，减少了命名冲突

###如何实现模块化
1.将变量写在函数里面
2.将变量使用对象进行管理
3.使用闭包

###模块化规范
1.浏览器端规范
amd：(https://github.com/amdjs/amdjs-api/wiki/AMD) 代表：require.js
cmd: (https://github.com/cmdjs/specification/blob/master/draft/module.md) 代表：sea.js
2.服务器端规范
commonjs:(http://wiki.commonjs.org/wiki/CommonJS) 代表：node.js
ECMAScript2015(ES6)模块化规范

##seajs使用

###cmd规范特性
1.一个模块就是一个单独的文件
2.每个模块都是属于define关键字的回调函数被调用，每个模块都是一个单独的作用域
3.预加载、懒执行(预加载就是要先把所有的模块加载完成，懒执行就是需要执行那个模块才会执行那个模块)

###如何加载模块
1.在html文件中引入sea.js库(假设html同级的js文件夹中存放在js文件)
2.在页面script中使用seajs.use加载主模块，起名一般叫main.js

面向对象笔记

##面向对象性能优化

 1.有关for循环：
 当循环取值时，每次都要重新获取数组长度，会降低代码，尤其是操作HTMLCollections，需要实时查询DOM ，而DOM操作一般是比较昂贵的
 ，因此，我们在循环取值时可以缓存数组或集合的长度，这样只需查询一次，可以提高代码运行效率；
 例：for (var i = 0, max = myarray.length; i < max; i++) { 
     // 使用myarray[i]做点什么}

 2.能力检测：
 在框架加载时判断浏览器是否支持某些功能，保存在变量中，以后使用时判断变量即可，减少原型链的搜索，可以提高性能。
 判断浏览器是否支持某些方法可以进行方法能力检测或方法定义检测。
  方法定义检测：浏览器的内置方法都有类似的结构；{ [native 可以用正则表达式: /\{\s*\[native/ 判断
 例：function querySelectorAll() { [native code] }
 /\{\s*\[native/.test(document.querySelectorAll);   true支持/false不支持

 3.检测是否含有该类名：  cName是要匹配的类名
 将元素和匹配的类名的类名前后都加上一个空格，这样元素中的每一个类名前后都是以空格分开的，如果元素含有要匹配的类名，通过indexOf方法可返回
 一个大于-1的索引，如果没有，就返回-1；
 例：var eles = document.getElementsByTagName('*'),i;
 for(i = 0; i < eles.length; i++){
     if((' ' + eles[i].className + ' ').indexOf(' ' + cName + ' ') >-1){
         alert('匹配成功');
     }else{
         alert('匹配失败');
     }
 }

 4.Object.prototype 的成员
 1) constructor
 2) hasOwnProperty 判断该属性是否为自己提供
 3) propertyIsEnumerable 判断属性是否可以枚举
 4) isPrototypeOf    判断是否为原型对象
 5) toString, toLocaleString, valueOf

 5.documentfragment对象表示内存节点,在页面上不占位置
 创建节点通过  var dfg = document.createDocumentFragment();

 6.添加事件监听
 标准API   dom.addEventListener('click',function(e){...});
 IE低版本  dom.attachEvent('onclick',function(e){...});   该事件执行顺序是倒着的，先添加的会后执行

 7.获取原生的所有事件的方法
 var dv = document.createElement('div'),results = [];
 for(var k in dv){if(k.slice(0,2) == 'on'){results.push(k)}}
 所有的事件名称都保存在results中了。

 8.实现快速继承的方法
var o = {};
var obj = Object.create(o);  以o为原型创建出来的对象赋值给obj
