###angular简介
angular是一款非常优秀的前端高级框架，其核心通过指令扩展了html，通过
表达式绑定数据到html。在angular中是不推荐使用dom操作的，因为angular倡导
以测试驱动开发，在service或controller中出现了DOM操作，那么就意味着测试是
无法通过的。使用框架可以轻松实现SPA(单页应用)应用程序。

###angular安装
1.暴力安装：直接复制一个angular.js
2.npm方式：npm install angular@1.5.2安装指定版本的
3.bower 方式 ‘npm install angular’

###指令
1.ng-app指令告诉angularjs应用当前这个元素是根元素，所有angularjs都
必须有一个根元素，当有多个ng-app指令，只有第一个会使用
2.ng-model把对应的值显示到页面中：例ng-model=”var”
3.ng-click用来注册点击事件：例ng-click=”add()”
4.ng-init用于初始化某个变量：例ng-init=”var= ‘小明’”

###什么是MVC思想和MVVM思想
M：Model 模型：数据存储，一些业务逻辑
V：View 视图 ：用来展示数据
C Controller 控制器：调度业务逻辑
VM ViewModel

###angular核心特性
1.模块化 angular.module()
angular.module(‘myApp’,[])
第一个参数是模块的名字
第二个是一个数组，数组的元素是该模块所依赖的其他模块的名字，若无依赖，传空数组
2.控制器controller
angular.moule(‘myApp’,[]).controller(‘myController’,function($scope){}));
第一个参数是控制器的名字
第二个参数是一个回调函数，传入参数$scope($scope对象在angularjs中充当数据模型的作用，也就是一般MVC框架中Model的角色，但与通常意义上的数据模型不一样，
因为$scope并不处理和操作数据，他只是建立了视图和html之间的桥梁，让视图和controller之间可以友好的通讯)
2.双向数据绑定
数据模型的值发生改变，就会导致页面值的改变，页面值的改变就会导致数据模型值
的改变，这种相互影响的关系叫双向数据绑定。
