canvas

##canvas是一个画布，默认大小是300*150

如果想改变大小必须通过html的width，height属性增加像素改变大小，不能直接通过css设置，css设置会直接拉伸像素大小，并不能改变画布大小

##想要在canvas上绘图，必须得到canvas绘图工具

var ctx = canvas.getContext("2d");   2d 平面绘图    webgl 3d绘图
ctx.moveTo(100,100);//将绘图工具定点到100,100位置
ctx.lineTo(200,100);//从刚才位置绘制到哪里
ctx.stroke();描边绘图，将绘制的图形呈递出来

##ctx.fill()

调用fill方法，会将所有的点连接起来，即使没有结束的那条线，也会构成一个封闭的结构

###ctx.closePath() 

闭合路径方法，可以将结束点与开始点连接起来
ctx.lineWidth = num：用来设置线条的宽度  num是宽度

###ctx.beginPath() 

在canvas中开启新的内容，不影响上一次绘制

###虚线

ctx.setLineDash(arr)   arr中存在着实线部分和虚线部分

###颜色

ctx.strokeStyle= 'red';  设置描边颜色
ctx.fillStyle= 'red';  设置填充颜色

##绘制矩形

如果不设置style，不需要设置beginPath
ctx.rect(x,y,width,height);   虚描，需要描边  x,y起始点
ctx.strokeRect(x,y,width,height);   直接画矩形
ctx.fillRect(x,y,width,height);   填充矩形
ctx.clearRect(x,y,width,height); 清除绘制的矩形区域,橡皮擦功能
ctx.clearRect(0,0,cas.width,cas.height);清除整个画布
cas.width = cas.width;  相当于把画布刷新，也可以清除整个画布

##绘制圆弧

ctx.arc(x,y,r,startAngle,endAngle,clockWise);   需要描边呈现
x,y圆心坐标;r圆弧半径；startAngle圆开始角度；endAngle圆结束角度(需要转换为弧度) ；Math.PI = 180度   radian = angle * Math.PI /180；

##绘制文本

ctx.strokeText(文本，x,y);  文本字符串的左下角在x,y坐标处；文本内容空心
ctx.fillText(文本，x,y);   实心文本内容
ctx.font = '30px 黑体';   设置字体样式  大小、字体
ctx.textAlign水平对齐 取值left center right ； start end 默认为start  
ctx.textBaseline基线对齐 取值 top  middle bottom alphabetic(基线对齐)
ctx.measureText(文本).width；获取当前文本的宽度

##图片绘制

1.创建图片对象  var  img = new Image(); 
2.img.src = '...'
3.1 ctx.drawImage(img,x,y);  将图片绘制到x,y的位置
3.2 ctx.drawImage(img,x,y,width,height); 将图片绘制到矩形区域，
进行计算，等比缩放，
3.3 ctx.drawImage(img,sx,sy,sw,sh,x,y,w,h); 将图片的某个部分
绘制到canvas中，一般截取的部分宽高与绘制的宽高一致
drawImage不仅可以绘制图片，也可以绘制其他其它标签，例video、canvas等

##变换

变换操作的是坐标轴，当坐标轴发生了变化以后，坐标轴内的图形也会跟着变化
ctx.translate()  平移变换
ctx.rotate()  旋转变换
ctx.scale()   伸缩变换

##canvas状态

canvas中，凡是设置属性，都会延续到下一次修改。
ctx.save()  将当前状态保存   代码开始使用
ctx.restore()  将保存的状态恢复   代码结束使用
save 和 restore搭配使用，可以使上一次操作不影响下一次的
