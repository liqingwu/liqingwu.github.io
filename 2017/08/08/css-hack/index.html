css-hack

###左侧浮动,右侧自适应
在Firefox中,不支持不给外层盒子设宽度的方法,需要给整体盒子设宽度.

###能使字体变细的css
-webkit-font-smoothing: antialiased; 加在body中

###在点击FireFox a标签的时候会出现虚线框
解决方法:a:focus{
outline:none;
}

###ie bug
input 输入框会有叉号
input::-ms-clear, input::-ms-reveal{
display: none;
}

###正常开发的pc网站如何在移动端能不错乱显示
body min-width: 1180px; //版心宽度
